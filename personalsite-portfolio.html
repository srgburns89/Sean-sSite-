Recipe Site

This was the first introduction project into html.

Recipe Project Goals

  • Use headings (h1, h2, etc) for the title, and headings for different sections
  • Apply some very basic styling to the body element (change the background color, choose a font, maybe set a max-width to prevent the text from being too long)
  • Use unordered and ordered lists for the ingredients and directions
  • Use paragraphs as necessary
  • Avoid using div or span elements unless there is no better semantic tag available
